# Documentation

This directory contains visual documentation for the Zecrone platform.

## Directory Structure

```
docs/
├── screenshots/        # Application screenshots
├── diagrams/           # Architecture and flow diagrams
└── README.md           # This file
```

## Adding Screenshots

Place application screenshots in the `screenshots/` directory. Recommended naming convention:

### Web Application
- `dashboard.png` — Main dashboard with KPIs and charts
- `invoices-list.png` — Invoice list view
- `invoice-create.png` — Invoice creation form
- `invoice-payment.png` — Partial payment dialog
- `quotes.png` — Quote management
- `customers.png` — Customer list
- `customer-detail.png` — Customer profile with current account
- `suppliers.png` — Supplier management
- `products.png` — Product catalog with variants
- `sales-orders.png` — Sales order processing
- `purchase-orders.png` — Purchase order management
- `cash-bank.png` — Cash register and bank accounts
- `cheques.png` — Cheque lifecycle tracking
- `cheque-endorsement.png` — Cheque endorsement (ciro) flow
- `stock-management.png` — Stock movements and balances
- `stock-counts.png` — Physical inventory count
- `branches.png` — Branch management
- `branch-transfer.png` — Inter-branch stock transfer
- `reports-pnl.png` — Profit & Loss report
- `reports-cashflow.png` — Cash flow report
- `reports-aging.png` — Aging analysis
- `budgets.png` — Budget planning and progress
- `exchange-rates.png` — TCMB exchange rates
- `admin-users.png` — User management panel
- `admin-roles.png` — Role permission configuration
- `superadmin-tenants.png` — SuperAdmin tenant management
- `superadmin-licenses.png` — License management
- `trial-registration.png` — Self-service trial signup
- `dark-mode.png` — Dark theme example
- `i18n-en.png` — English language view

### Mobile Application

## Adding Diagrams

Place architecture diagrams, ER diagrams, and flow charts in the `diagrams/` directory. Note that the main ARCHITECTURE.md uses Mermaid diagrams which render natively on GitHub.
