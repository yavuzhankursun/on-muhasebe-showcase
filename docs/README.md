# Documentation

This directory contains screenshots, diagrams, and visual documentation for the Ön Muhasebe platform.

## Directory Structure

```
docs/
├── screenshots/        # Application screenshots
├── diagrams/           # Architecture and flow diagrams
└── README.md           # This file
```

## Adding Screenshots

Place application screenshots in the `screenshots/` directory. Recommended naming convention:

- `dashboard.png` - Main dashboard view
- `sales-order.png` - Sales order creation
- `invoice-detail.png` - Invoice detail view
- `stock-management.png` - Stock management page
- `cheque-tracking.png` - Cheque lifecycle view
- `admin-roles.png` - Role management panel
- `mobile-view.png` - Mobile responsive view

## Adding Diagrams

Place architecture diagrams, ER diagrams, and flow charts in the `diagrams/` directory.
